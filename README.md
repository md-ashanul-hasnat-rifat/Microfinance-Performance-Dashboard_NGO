# 📊 Executive Overview Dashboard (Power BI Project)

## 📌 Project Overview
This project presents an **Executive Overview Dashboard** built in Power BI to monitor and analyze key financial and operational metrics across different regions, districts, and programs.

The dashboard is designed to support **data-driven decision-making** by providing a high-level summary along with detailed insights.

---

## 🎯 Objectives
- Track overall **collection and disbursement performance**
- Monitor **overdue amounts and risk levels**
- Analyze **client acquisition trends**
- Compare **regional and program-based performance**
- Provide actionable insights for management

## 📈 Dashboard Features

### 1. Executive Summary Cards
- Monthly comparison with growth indicators  
- Quick overview of business performance  

### 2. Trend Analysis
- Monthly collection trends  
- Identify peak and low-performing periods  

### 3. Regional Performance
- Disbursement comparison across regions:
  - West  
  - South  
  - East  
  - North  
  - Central  

### 4. Program-wise Distribution
- Overdue breakdown by:
  - Microfinance  
  - Livelihood  
  - Health  
  - Education  

### 5. Target vs Actual Performance
- Collection achievement: **113% of target**  
- Client acquisition tracking  

### 6. Detailed Data Table
- Branch-level insights including:
  - Collection & Disbursement  
  - Overdue & Rate  
  - District & Region  
  - Program & Gender

## 🧩 Data Model
- Star schema approach used  

**Fact Table:**
- Transactions (Collection, Disbursement, Overdue)

**Dimension Tables:**
- Date  
- Branch  
- Region  
- Program  
- Client Info

## 📌 Key Insights
- Collection performance exceeded target (**113% achievement**)  
- Overdue rate is **11.14%**, indicating moderate risk  
- Majority of overdue comes from **Microfinance sector**  
- West region leads in disbursement  
- New client growth is relatively low compared to target

## 🚀 How to Use
1. Open the `.pbix` file in Power BI  
2. Use slicers to filter:
   - Gender  
   - District  
   - Region  
   - Month & Year  
3. Explore KPIs and trends interactively
